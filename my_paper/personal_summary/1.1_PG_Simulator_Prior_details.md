# Simulator Prior Details

## Intuition

The simulator is the scientific forward model: it links hidden parameters $\theta$ to observable data $x$, and the resulting simulations must be similar enough to the real observation for the posterior to be meaningful (Guide, p5:L38-L46).

SBI only requires that the simulator can be run forward, $x \sim p(x\mid\theta)$; it can treat the simulator as a black box without likelihoods, gradients, or access to internals (Guide, p6:L2-L12).

The prior $p(\theta)$ encodes knowledge about plausible parameter values before observing $x_o$, and standard SBI samples parameters from this prior to build the simulation dataset (Guide, p6:L14-L18).

Prior design is scientific, not neutral by default: uniform bounded priors are common and convenient, but the bounds and scale still encode assumptions (Guide, p6:L22-L29).

The main danger is misspecification: if $x_o$ lies outside the range generated by the simulator and prior, neural SBI can return erratic or spurious posteriors rather than simply becoming uncertain (Guide, p6:L37-L43; p7:L1-L6).

Prior width is a trade-off: too narrow can exclude the observation, while too broad wastes simulations in irrelevant regions and can reduce accuracy for a fixed simulation budget (Guide, p7:L7-L18).

## Mathematical Framework

Define the forward model as

$$
M: \theta \mapsto x,\qquad x \sim p_{\text{sim}}(x\mid\theta).
$$

The SBI training distribution is the joint

$$
p_{\text{train}}(\theta,x)=p(\theta)p_{\text{sim}}(x\mid\theta),
$$

with $\theta\sim p(\theta)$ and $x\sim p_{\text{sim}}(x\mid\theta)$ (Guide, p4:L6-L10; p6:L15-L17).

A useful validity condition is prior-predictive coverage:

$$
x_o \in \operatorname{support}\left(p_{\text{train}}(x)\right),
\qquad
p_{\text{train}}(x)=\int p_{\text{sim}}(x\mid\theta)p(\theta)\,d\theta.
$$

The guide recommends checking this by sampling from the prior, running the simulator, and visually or statistically comparing simulated data to the observation (Guide, p7:L21-L32; p13:L8-L17).

If there is no parameter setting under the simulator-prior pair that can generate $x_o$, then the conditioning event is outside the learned joint distribution and $q_\phi(\theta\mid x_o)$ is extrapolation rather than posterior inference (Guide, p6:L37-L43; p7:L1-L6).

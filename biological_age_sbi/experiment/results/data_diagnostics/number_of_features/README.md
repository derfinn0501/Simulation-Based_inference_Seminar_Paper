# Number Of Features

Feature-set diagnostics where each observation design is trained at the same
small simulation budgets: 2, 8, and 32 batches.

Each row in the result CSV is one trained network for one feature set and one
training-simulation budget. The plots compare training-loss convergence,
training loss versus simulations, Pearson `r` versus simulations, and q10-q90
width as a first certainty proxy.

# SBI Amortization Study

Research notes for understanding how simulation-based inference uses amortized learning, when amortization is a core advantage, and when SBI can be preferable to methods such as MCMC.

## Structure

- `notes/`: conceptual notes and working questions.
- `literature/`: paper summaries and source notes.
- `comparison_framework/`: criteria for deciding whether a problem is worth solving with SBI.
- `mini_experiments/`: small empirical checks comparing amortized SBI with non-amortized inference baselines.

The goal is to build a decision framework, not to argue that SBI is always better.

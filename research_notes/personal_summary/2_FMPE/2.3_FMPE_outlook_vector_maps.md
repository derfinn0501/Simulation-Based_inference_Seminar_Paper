# FMPE Outlook: Uses Of Vector Maps

## Intuition

FMPE does more than return posterior samples. It learns a conditional vector map that describes how probability mass moves from a simple base distribution into the posterior. This gives a richer object than a black-box sampler: the learned vector field contains geometric information about the posterior (FMPE, p3:L47-L53; FMPE, p10:L1-L18).

One outlook is to use these vector maps to understand posterior geometry. The trajectory from base noise to posterior sample may reveal where the posterior has sharp boundaries, multiple modes, or narrow correlated directions. This connects naturally to posterior analysis in the practical guide, where correlations, compensation mechanisms, and conditional structure are central (PG, p15:L16-L30; PG, p16:L4-L8).

A second outlook is simulation or experiment design. If a vector field is hard to learn in some regions, or if different models disagree about the vector field, this can signal where additional simulations would be useful. This connects naturally to BO-guided simulator-budget allocation: BO can optimize design variables $\psi$, while FMPE supplies diagnostics or uncertainty signals from the learned flow field.

A third outlook is model checking. Because FMPE can provide density access and mass-covering behavior, it can be paired with posterior predictive checks, coverage tests, and importance sampling corrections in scientific applications (FMPE, p5:L24-L34; FMPE, p9:L156-L160; PG, p13:L22-L34; PG, p14:L9-L37).

## Mathematical Framework

The learned object is a conditional vector field

$$
v_{t,x}(\theta).
$$

For each observation $x$, this field defines a deterministic transport:

$$
\theta_0 \sim q_0(\theta),
\qquad
\frac{d\theta_t}{dt}=v_{t,x}(\theta_t),
\qquad
\theta_1 \sim q_\phi(\theta\mid x).
$$

This can be interpreted as a learned map from simple uncertainty to posterior uncertainty.

Possible uses:

### 1. Posterior Geometry

The path

$$
\{\theta_t: t\in[0,1]\}
$$

can be studied to identify difficult posterior structure: strong curvature, bottlenecks, multimodality, or regions where trajectories diverge. These are geometric analogues of posterior correlations and non-identifiabilities.

### 2. Active Simulation Design

Use vector-field uncertainty as a design score. For an ensemble of FMPE models,

$$
u(\psi)
=
\mathbb{E}_{\theta,x,t}
\left[
\operatorname{Var}_m
\left(
v^{(m)}_{t,x}(\theta)
\right)
\right],
$$

where

$$
\theta\sim p(\theta),
\qquad
x\sim p_{\text{sim}}(x\mid\theta,\psi).
$$

BO could then choose $\psi$ where the posterior transport is uncertain, without directly optimizing the physical parameter $\theta$.

### 3. Posterior Predictive Design

Because the vector field generates posterior samples, one can combine FMPE with posterior predictive checks:

$$
\theta^{(m)} \sim q_\phi(\theta\mid x_o,\psi),
\qquad
\tilde{x}^{(m)}
\sim
p_{\text{sim}}(x\mid\theta^{(m)},\psi).
$$

Designs $\psi$ can then be scored by how well posterior predictive simulations match relevant features of $x_o$.

### 4. Importance Sampling And Correction

If likelihood evaluation is possible, a mass-covering FMPE posterior can be used as a proposal for importance sampling:

$$
w(\theta)
\propto
\frac{p(x_o\mid\theta)p(\theta)}
{q_\phi(\theta\mid x_o)}.
$$

The FMPE paper explicitly notes that mass coverage is useful because it reduces the risk of missing plausible posterior regions and enables importance-sampling refinement when the forward likelihood is available (FMPE, p5:L24-L34; FMPE, p9:L156-L160).

## Outlook Summary

The vector map can be used as:

1. a posterior sampler,
2. a density estimator,
3. a geometric description of posterior transport,
4. an uncertainty signal for active simulation design,
5. a proposal distribution for importance sampling,
6. a diagnostic object for where posterior learning is difficult.

This is the deeper value of FMPE for your project: it does not only improve the posterior estimator, it creates a structured object that BO or validation tools can inspect.

# Experiment Results

This folder stores result bundles by approach number, so the project history is visible.

Naming convention:

```text
approach_<number>_<short_description>/
```

Current folders:

```text
approach_4_1_active_design_snapshot/
approach_4_2_fmpe_quality_check/
```

`approach_4_1_active_design_snapshot` keeps the earlier BO-vs-random active-design outputs.

`approach_4_2_fmpe_quality_check` is for standalone FMPE diagnostics under random design. It asks whether FMPE is good enough before BO is evaluated.

# Project Summary

This folder holds the stable project summary.

Use it for:

- project context
- research style
- current state
- stable learnings

Root-level files are reserved for operating workflow, backlog, decisions, and
agent instructions.

## Files

- `PROJECT_CONTEXT.md`: topic, scope, and current experimental framing
- `CURRENT_STATE.md`: concise summary of current progress
- `RESEARCH_STYLE.md`: preferred thinking and explanation style
- `LEARNINGS.md`: stable findings from completed work
